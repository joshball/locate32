Arrange Icons (View menu)

The commands in this menu can be used to arrange and sort files.

The menu offers the following commands:

by Name Sort results by name.
by Folder Sort results by folder.
by Type Sort results by file type.
by Size Sort results by file size.
by Date Sort results by date.
by Extension Sort results by file extension.
Auto Arrange Arranges icons automatically.
Align to Grid Aligns icons to grid.
