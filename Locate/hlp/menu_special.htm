Special menu (File menu or context menus for the selected items)

The Special menu offers the following commands:

Copy path to clipboard Copies the path of the selected files to the clipboard.
Copy short path to clipboard Copies the path of the selected files to the clipboard in short name format.
Change filename Shows dialog which can be used to rename file.
Change case for filename Changes case for names of selected files.
Show total size of selected files Calculates the total size of selected files.
Update file information Reads file information from file.
Compute MD5 sums Computes MD5 sum for selected files.
Compute MD5 sums for equal size files Computes MD5 sum for equally sized files.
Remove deleted files Remove deleted files from the result list.
